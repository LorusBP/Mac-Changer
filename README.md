# Mac-Changer
A Simple Linux Mac-Changer

USAGE


usage: Mac-Changer.py [-h] [-i INTERFACE] [-m MAC]
                                                                                                                                                                                                                                           
optional arguments:                                                                                                                                                                                                                        
  -h, --help            show this help message and exit                                                                                                                                                                                    
  -i INTERFACE, --interface INTERFACE                                                                                                                                                                                                      
                        Interface that is being used                                                                                                                                                                                       
  -m MAC, --mac MAC     New Mac Address
